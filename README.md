# Robot Boxeur

Application Java réalisée pour s'entraîner à la programmation orientée objet et à la création d'interfaces graphiques.

## Technologies

- Java
- JavaFX
- Programmation Orientée Objet

## Fonctionnalités

- gestion des tours de combat
- barres d'énergie
- logique d'attaque et défense
- interface graphique

## Objectif du projet

Ce projet m'a permis de pratiquer la programmation orientée objet et la création d'interfaces en JavaFX.
